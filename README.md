## 使用说明

### 一、获取运行所需的 Cookies

#### 方法1（快速获取）

- 打开哔哩哔哩网页端登录好
- 新建一个浏览器页面,打开开发者模式,复制链接`http://account.bilibili.com/site/getCoin`黏贴打开
- 复制`cookie`填入配置文件`config/config.py`的`COOKIE_LIST `变量中![](/doc/ck.png)



#### 方法2（多账号，无痕模式获取）
#### Example
![](doc/WX20230616-213234@2x.png)

1. **获取 Bilibili Cookies**
2. 浏览器无痕模式打开并登录 [bilibili 网站]()。不使用无痕模式则退出账号会cooikies会失效
3. 按 F12 打开 「开发者工具」 找到 应用程序/Application -\> 存储 -\> Cookies
4. 找到 `bili_jct` `SESSDATA` `DEDEUSERID` 三项，并复制值，后面需要用到。

![image-20230614111857210](doc/image-20230614111857210.png)

5.将上面复制值填入r"DedeUserID=xxxx; SESSDATA=xxxx; bili_jct=xxxx"，并填入配置文件`config/config.py`的`COOKIE_LIST `中。多个账号用，分割。形成一个列表

![image-20230614113231077](doc/image-20230614113231077.png)

#### 2.3 pushplus推送

1. 访问PushPlus的[官方网站](https://www.pushplus.plus/)并注册一个账号。
2. 在注册成功后，登录PushPlus的控制台。
3. 在控制台的左侧菜单中，选择“一对一推送”或“一对多推送”，根据您的需求选择推送类型。
4. 在推送设置页面中，将看到一个名为“Token”的字段，这就是PushPlus的推送key。
5. 获取后填入置文件`config/config.py`的中，并将PUSH_OR_NOT改为True

![image-20230614102656471](doc/image-20230614102656471.png)



### 三、部署

#### 3.1 使用 阿里云部署

目前阿里云函数调用费用很低，仅仅部署本项目一个月金额非常低，到后面会抹零头，最终支付0元。等于白嫖。

1. 登录登录[阿里云函数计算](https://account.aliyun.com/login/login.htm?oauth_callback=https%3A%2F%2Ffcnext.console.aliyun.com%2Foverview&lang=zh)支付宝扫码登录，进入函数计算控制台。创建服务

2. 在左侧菜单中选择“函数服务”，然后单击“创建函数”按钮。

3. ![image-20230614103557943](doc/image-20230614103557943.png)

4. 在“创建函数”页面中，选择“内置运行时”选项卡，然后选择“Python 3.6”作为运行环境。

5. ![image-20230614103634590](doc/image-20230614103634590.png)

6. 在“函数配置”中，输入函数名称、函数描述和函数入口，即example/tencent_cloud.main_handler。

7. 在“高级配置”中，设置执行超时时间为600秒。

8. 单击“下一步”，进入“函数代码”页面。

9. 在“函数代码”页面中，通过文件夹或者zip上传代码，注意上传完成后，根目录必须是main.py文件所在目录。

10. 单击“保存并部署”按钮，等待函数部署完成。

11. ![image-20230614104028085](doc/image-20230614104028085.png)

    

12. 部署完成后，您可以单击“测试”按钮来测试函数是否正常工作。

13. 如果测试通过，则可以使用函数的触发器来触发函数的执行。您可以选择阿里云定时触发器每天触发，自动执行

#### 3.4 使用 windows平台部署

使用该方式要求电脑每天都使用，并且保证每天电脑有一段时间是开机状态。适合使用电脑上班的人使用。

##### 1、测试是否正常使用

确认已经安装了 python

- 安装依赖`pip install -r requirements.txt`

下载该项目

打开命令行窗口或终端，进入main.py文件所在目录

`python main.py`，执行。测试是否有问题

执行命令后，程序会自动执行，并在控制台输出结果。

##### 2、每天定时自动执行

如果需要每天定时执行签到任务，可以使用 Windows 系统的计划任务程序来实现：

在main.py所在目录下新建一个bat文件,填入

```shell
cd /d %~dp0
python main.py
```

此电脑，右键

打开“任务计划程序”，创建一个新任务



![image-20230610013850350](doc/image-20230610013850350.png)

在“触发器”选项卡中，设置每天定时执行任务的时间

在“操作”选项卡中，添加一个新操作，选择“启动程序”，添加刚刚的脚本

![image-20230610013953579](doc/image-20230610013953579.png)

保存任务后，系统会在每天设定的时间自动执行任务。
